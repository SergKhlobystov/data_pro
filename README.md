# GO Data Analyst Pro

[![Version](https://img.shields.io/badge/version-2.7.1-blue.svg)](https://github.com/serg-heavybeard/go-data-analyst)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**GO Data Analyst Pro** — це професійний інструмент для генерації реалістичних синтетичних даних, створений спеціально для аналітиків, які хочуть відточити свої навички на практиці.

---

## 🎯 Суть проекту (The Essence)

Головна проблема початківців та навіть досвідчених аналітиків — пошук якісних даних для тренувань. Часто доступні датасети або занадто ідеальні, або не відповідають реальним бізнес-сценаріям. 

**GO Data Analyst** вирішує цю проблему, надаючи можливість миттєво створювати таблиці з даними, які імітують реальну поведінку користувачів, транзакції та продуктові метрики.

## 💡 Головні ідеї при створенні (Core Ideas)

1. **Реалістичність понад усе**: Дані не просто випадкові. Вони мають логічні зв'язки (наприклад, залежність рейтингу від активності або типи підписок залежно від пристрою).
2. **Градація складності**: 
   - **Easy**: Чисті дані для базових SQL-запитів та Excel.
   - **Medium**: Дані з невеликими аномаліями та додатковими вимірами (демографія, канали).
   - **Hard**: "Брудні" дані, складні моделі монетизації (SaaS/Freemium) та міжнародні ринки, що потребують глибокого очищення та складного аналізу.
3. **Швидкість та доступність**: Ніяких реєстрацій чи складних налаштувань. Один клік — і CSV файл у вас на комп'ютері.
4. **Сучасний UX/UI**: Мінімалістичний дизайн у стилі Apple, що не відволікає від головного — роботи з даними.

---

## 🚀 Функціонал

- **Генерація до 20,000 рядків** за один раз.
- **3 рівні складності** з різною структурою (від 10 до 20 стовбців).
- **Мультимовність**: Підтримка української, англійської та російської мов.
- **Темна та світла теми**: Для комфортної роботи в будь-який час доби.
- **Експорт у CSV**: Формат, готовий до імпорту в SQL, Python (Pandas), Tableau або Power BI.

## 🛠 Технологічний стек

- **React 18** + **TypeScript**
- **Tailwind CSS 4** (для стилізації)
- **Framer Motion** (для анімацій)
- **Lucide React** (іконки)
- **Vite** (збірка)
